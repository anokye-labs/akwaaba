# Phase 4: Example .NET Application

**Goal:** Demonstrate agents maintaining real code.

## Features

1. **dotnet-project-init** - .NET console application
2. **dotnet-specs** - Spec-kit specifications
3. **dotnet-tests** - Test infrastructure
4. **scripts-issue-workflow** - Issue → branch → PR scripts
5. **scripts-agent-utils** - Agent development helpers

## Dependencies

- Phase 1 (structure)
- Phase 3 (agent-runner-common for scripts)

## Success Criteria

- .NET app compiles and runs
- Specs define agent work
- Tests pass
- Automation scripts work
- Agents can modify the app

## Estimated Effort

Medium - straightforward implementation, 2-3 sessions
