## Overview

This guide will help you replicate the Anokye-Krom System pattern in your own repository.

## Prerequisites

Ensure you have the following installed:

- **GitHub CLI** (`gh`) version 2.0 or later
- **.NET SDK** version 8.0 or later
- **PowerShell** version 7.0 or later

Verify your installations:

```bash
gh --version
dotnet --version
pwsh --version
```
